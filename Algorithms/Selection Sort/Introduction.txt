Time Complexity : O(n^2)
Space Complexity : O(1)
Selection sort is not stable.
Selection sort is not Adaptive.
Max-Swap : n-1
Min-Swap : 0

Step 1 : Find the minimum element in the array and swap it with the first element.
Step 2 : Find the minimum element in the array starting from the second element and swap it with the second element.
Do above steps until we reach end of an array.
